SOUND FILES FOR OFFERS PAGE
============================

Please add the following sound files to this directory (assets/sounds/):

1. "Egypt.mp3" - Sound file for Egypt tours (plays on hover with fade in/out)
2. "India.mp3" - Sound file for India tours (plays on hover with fade in/out)
3. "japan.mp3" - Sound file for Japan/Shibuya tours (plays on hover with fade in/out)

IMPORTANT: The file names must match exactly as shown above (case-sensitive).

The sound files will automatically play when users hover over the corresponding offer cards with smooth fade in/out effects.

Supported formats: MP3, OGG, WAV (MP3 recommended for best compatibility)

Note: If sound files are not available, the page will still work but without hover sounds.
Check the browser console (F12) for error messages if sounds don't play.
